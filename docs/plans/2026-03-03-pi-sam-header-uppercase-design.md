# pi-sam 헤더 로고 대문자 전환 디자인

## 배경
기존 compact 블록형 로고의 소문자 표현은 곡선이 많아 가독성이 떨어진다. 사용자 요청에 따라 로고 텍스트를 대문자(`PI-SAM`)로 전환해 시인성을 높인다.

## 요구사항
- 로고 텍스트: `PI-SAM` (대문자)
- 크기: compact 유지 (3줄, 폭 20~28자)
- 스타일: 블록형 (`█`, `▀`)
- 하단 메타 라인(`/help · /model · /resume  v<version>`) 유지

## 접근안 비교
1. 로고 3줄 문자열만 교체 (선택)
2. 테스트에 대문자 자체를 강하게 고정
3. 대/소문자 프리셋 옵션화

## 선택
- 1번 선택. 변경 범위 최소, 리스크 최소, 요구사항 충족.

## 설계
- `src/extensions/branding-header.ts`의 상단 3줄만 대문자형 패턴으로 교체
- `src/extensions/branding-header.test.ts`의 기존 compact 제약은 유지
- 동작/색상/메타라인/라인수 규칙은 유지
